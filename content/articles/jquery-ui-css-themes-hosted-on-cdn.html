(: #:title:# ~ Links to jQuery UI CSS themes hosted on Google's CDN :) (: #:head:# ~ :) (: markdown ~ ## Links to jQuery UI CSS themes hosted on Google's CDN July 7, 2010 It's well known that Google hosts several [popular javascript libraries](http://code.google.com/apis/ajaxlibs/) on their public CDN. A lesser know fact is that google also hosts an number CSS themes for jQuery UI (both [v1.7.3](http://ajax.googleapis.com/ajax/libs/jqueryui/1.7.3/jquery-ui.js) and [v1.8.7](http://ajax.googleapis.com/ajax/libs/jqueryui/1.8.7/jquery-ui.js )). For whatever this remains undocumented. In the past, I've had a hard time finding them, so I decided to help myself out (and maybe others) and put a list together. ### jQuery UI 1.8.7 CSS Themes: :)
